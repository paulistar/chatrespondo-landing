# ChatRespondo Landing

Landing page de vendas em [chatrespondo.com](https://chatrespondo.com).

## Stack

- Vite + Tailwind CSS
- HTML estático (multi-page: index, privacidade, termos)
- Deploy: Docker + nginx no EasyPanel

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## CTAs

Todos os botões de trial apontam para `panel.chatrespondo.com/register` com UTMs preservados.
